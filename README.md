#  Book Rental System (EMT Project)

A full-stack book rental application built with **Spring Boot** and **React**.  
The system allows users to manage books, authors, and countries with full CRUD functionality and secure authentication.

---

##  Features

- Manage books, authors, and countries (CRUD operations)
- User authentication and authorization using **JWT**
- Role-based access control (User / Librarian)
- RESTful API built with Spring Boot
- PostgreSQL database integration
- Optimized data fetching using advanced JPA features
- React frontend with routing and modular components
- API integration using **Axios**
- Reusable UI components and modal dialogs

---

##  Tech Stack

### Backend
- Spring Boot
- Spring Security
- PostgreSQL
- JWT Authentication
- JPA / Hibernate

### Frontend
- React
- React Router
- Axios
- Material UI
